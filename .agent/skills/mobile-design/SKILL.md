# Mobile Design - Touch-First Principles

> **Curator:** AB ED&IA - Alexandre Belo | **Project:** Antigravity-Kit-Brabo

## Touch Targets

- Minimum 44x44 points (iOS)
- Minimum 48x48 dp (Android)
- 8px gap between targets

---

## Safe Areas

- Respect notches and home indicators
- Test on multiple device sizes
- Use platform safe area APIs

---

## Performance

- 60 FPS target
- Cold start < 3s
- Lazy load images
- Minimize network calls

---

## Platform Conventions

| iOS | Android |
|-----|---------|
| Back arrow left | Back arrow left |
| Tab bar bottom | Bottom navigation |
| Pull to refresh | Pull to refresh |
| Swipe gestures | Swipe gestures |

---

**Remember**: Mobile is not small web. Design for touch and context.
