# Plan Writing - Structured Planning

> **Curator:** AB ED&IA - Alexandre Belo | **Project:** Antigravity-Kit-Brabo

## When to Use

- Multi-file changes
- New features
- Refactoring
- Migrations

---

## Plan Template

```markdown
# Plan: [Feature Name]

## Objective
[What will be built]

## Scope

### Included
- [Feature 1]
- [Feature 2]

### Not Included
- [Out of scope]

## Tasks

### Phase 1: Setup
- [ ] **1.1** Task description
- [ ] **1.2** Task description

### Phase 2: Implementation
- [ ] **2.1** Task description

## Verification
- [ ] Tests passing
- [ ] Manual verification
```

---

**Remember**: Plans prevent scope creep and miscommunication.
