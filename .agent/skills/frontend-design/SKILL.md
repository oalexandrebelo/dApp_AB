# Frontend Design - UI Principles

> **Curator:** AB ED&IA - Alexandre Belo | **Project:** Antigravity-Kit-Brabo

## Core Principles

- **Consistency**: Same patterns everywhere
- **Hierarchy**: Visual importance order
- **Whitespace**: Let elements breathe
- **Feedback**: Respond to user actions

---

## Color

- Primary: Brand color for CTAs
- Neutral: Grays for text/background
- Semantic: Green=success, Red=error
- Contrast: 4.5:1 minimum for accessibility

---

## Typography

- 16px body minimum
- 1.5 line-height for readability
- Max 3 font sizes per page
- Font stack with fallbacks

---

## Spacing

Use 8px grid:
- 8, 16, 24, 32, 48, 64, 96

---

**Remember**: Design serves users, not aesthetics.
