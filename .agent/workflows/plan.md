# /plan - Create Implementation Plan

Use when: Planning before implementation.

## Flow

1. Analyze: Understand requirements
2. Break down: Create task list
3. Estimate: Time and effort
4. Document: Generate PLAN.md

## Examples

```
/plan implement JWT authentication
/plan migrate database to PostgreSQL
/plan refactor user module
```
