# /brainstorm - Idea Exploration

Use when: Need to explore options before deciding.

## Flow

1. Define: What problem to solve?
2. Explore: Multiple approaches
3. Evaluate: Pros and cons
4. Decide: Best path forward

## Examples

```
/brainstorm authentication strategies
/brainstorm database architecture
/brainstorm UI/UX for dashboard
```
